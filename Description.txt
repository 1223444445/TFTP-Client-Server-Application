description : The objective of this project was to implement a simple file transfer application based on the Trivial File Transfer Protocol (TFTP).
              The project supports transferring files between a client and server using UDP sockets. It follows the TFTP protocol specifications 
              for read and write requests, data packet transmission, acknowledgements, and error handling. The implementation ensures reliable file
              transfer by managing block numbers, retransmissions, and proper termination of file transfer sessions.
